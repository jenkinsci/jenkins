Se quest'opzione è abilitata, verranno salvati in Jenkins la data dell'utilizzo più recente di ogni token API e il numero totale di volte in cui questo è stato utilizzato. Ciò consente agli utenti di vedere se hanno dei token API non utilizzati od obsoleti che dovrebbero essere revocati.
Questi dati sono salvati all'interno dell'istanza di Jenkins e non saranno utilizzati per alcun altro scopo.
