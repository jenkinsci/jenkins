Det här alternativet konfigurerar den minsta mängden ledigt temporärt utrymme som önskas för att agenten ska fungera ordentligt, t.ex. "1.5 GB", "100KiB" etc. Om agenten visar sig ha mindre ledigt temporärt utrymme än denna mängd visas en varning på datorns översiktssida, men den kommer inte att göras frånkopplad.

Ange 0 för att inaktivera denna varning på denna agent.

Lämna tomt för att använda det globalt definierade värdet.
