Det här alternativet konfigurerar den minsta mängden ledigt diskutrymme som önskas för att agenten ska fungera ordentligt, t.ex. "1.5 GB", "100KiB" etc. Om agenten visar sig ha mindre ledigt diskutrymme än denna mängd kommer den att tillfälligt markeras som frånkopplad.

Ange 0 för att inaktivera denna kontroll på denna agent.

Lämna tomt för att använda det globalt definierade värdet.
