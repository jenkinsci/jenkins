Überwacht den freien Festplattenplatz im Temporärverzeichnis. Unterschreitet der freie Platz eine gewisse Schwelle, so wird der Agent offline genommen.

Java-Werkzeuge und Tests/Builds arbeiten häufig mit Dateien im Temporärverzeichnis und können daher unerwartete Ergebnisse liefern, wenn der Platz in diesem Verzeichnis erschöpft ist.

Präziser ausgedrückt, wird der freie Festplattenplatz auf der Partition ermittelt, die das Verzeichnis enthält, auf welches die Systemeigenschaft java.io.tmpdir verweist. Um herauszufinden, wo dieses Verzeichnis auf einem bestimmten Agent liegt, rufen Sie ${rootURL}/computer/SLAVENAME/systemInfo auf.
