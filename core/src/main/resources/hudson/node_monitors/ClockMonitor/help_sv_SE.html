Detta övervakar klockskillnaden mellan styrenheten och noderna. Medan själva Jenkins är i allmänhet kapabel att tolerera klockskillnader mellan system tenderar versionskontrollaktiviteter och distribuerad filåtkomst (t.ex. NFS, Windows-filresurser) att ha konstiga problem när inblandade system har olika klockslag.
Den markerar aldrig en agent som frånkopplad.

Ange NTP för att hålla klockorna synkroniserade.
