Det här alternativet konfigurerar den minsta mängden ledigt diskutrymme som önskas för att en agent ska fungera ordentligt, t.ex. "1.5GB", "100KiB" etc. Om en agent visar sig ha mindre ledigt diskutrymme än denna mängd kommer den att tillfälligt markeras som frånkopplad. Tröskelvärdet kan konfigureras individuellt för varje agent via en egenskap.

Ange 0 för att inaktivera denna kontroll förutom för agenter som uttryckligen konfigurerar tröskelvärdet.
