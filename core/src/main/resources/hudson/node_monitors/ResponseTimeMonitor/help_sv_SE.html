Detta övervakar nätverkets svarstid fram och tillbaka mellan styrenheten och agenten samt gör denna agent frånkopplad om den går över ett tröskelvärde upprepade gånger.

Detta är användbart för att upptäcka agenter som inte svarar eller andra nätverksproblem som täpper till kommunikationskanalen. Närmare bestämt skickar styrenheten ett kommando som inte gör någonting till agenten och kontrollerar hur länge det tar att få tillbaka resultatet från detta kommando.
