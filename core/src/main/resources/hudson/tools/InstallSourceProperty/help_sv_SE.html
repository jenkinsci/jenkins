Välj detta alternativ för att låta Jenkins installera verktyget åt dig på begäran.

Om du markerar detta alternativ kommer du ombes att konfigurera en serie "installationsprogram" för detta verktyg, där varje installationsprogram definierar hur Jenkins ska försöka installera verktyget.

För ett plattformsoberoende verktyg (som Ant) är det inte så logiskt att konfigurera flera installationsprogram för ett enda verktyg, men för ett plattformsberoende verktyg låter flera installationskonfigurationer dig köra ett annat konfigurationsskript beroende på agentmiljön.
