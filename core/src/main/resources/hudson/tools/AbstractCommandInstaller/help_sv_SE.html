Kör ett valfritt skalkommando för att installera verktyget. För Ubuntu, förutsatt att Jenkins-användaren är i /etc/sudoers : 

sudo apt-get --yes install openjdk-6-jdk

(Ange i det här fallet t.ex. /usr/lib/jvm/java-6-openjdk-i386 som hemkatalog.)

Ett annat exempel för att installera en äldre version av Sun JDK 6 för Linux (x86) kan du använda den föråldrade DLJ : 

bin=jdk-6u13-dlj-linux-i586.bin
if [ \! -f $bin ]
then
    wget --no-verbose http://download.java.net/dlj/binaries/$bin
    sh $bin --unpack --accept-license
fi

(Ange i det här fallet jdk1.6.0_13 som hemkatalog.)
