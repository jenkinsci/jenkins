Esegue un comando shell a scelta per installare lo strumento. Esempio su Ubuntu, assumendo che l'utente Jenkins sia elencato in /etc/sudoers: 

sudo apt-get --yes install openjdk-6-jdk

(In questo caso si specifichi ad es. /usr/lib/jvm/java-6-openjdk-i386 come directory home.)
