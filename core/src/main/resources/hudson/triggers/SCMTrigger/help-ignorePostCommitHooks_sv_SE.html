Ignorera förändringar som meddelas av hooks i versionskontrollen efter utcheckningar.

Detta kan vara användbart om du vill förhindra att vissa långvariga jobb (t.ex. rapporter) startar på grund av varje utcheckning, men ändå vill köra dem med jämna mellanrum om ändringar i versionskontrollen har inträffat.

Observera att det här alternativet också måste stödjas av versionskontrollens insticksprogram! Insticksprogrammet för Subversion stödjer detta sedan version 1.44.
