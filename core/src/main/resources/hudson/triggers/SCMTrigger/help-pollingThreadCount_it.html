Quest'opzione configura il numero massimo di thread concorrenti che sono utilizzati da Jenkins per eseguire il polling delle modifiche nel sistema di gestione del codice sorgente.

A seconda della connessione di rete, del servizio del sistema di gestione del codice sorgente e della configurazione di Jenkins potrebbe essere necessario aumentare o diminuire il numero di thread per ottenere dei risultati ottimali.
