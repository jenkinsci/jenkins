Доменное имя почтового сервера. Оставьте пустым для использования сервера по-умолчанию (обычно это тот сервер запущен на localhost).

Hudson использует JavaMail для отправки почты. JavaMail имеет возможность дополнительной настройки через системные свойства установленные в контейнере. Смотрите документ для полного списка параметров и их описаний.
