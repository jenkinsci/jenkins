Inställningselementet i filen settings.xml innehåller element som används för att definiera värden som konfigurerar Maven-körningen på olika sätt, som pom.xml , men bör inte buntas ihop till någon specifikt projekt eller distribueras offentligt. Detta inkluderar värden som den lokala kodlagringsplatsen, alternativa fjärrlagringsservrar och autentiseringsinformation.
Som standard finns det två platser där en fil som heter settings.xml kan finnas:
  • Installationen av Maven - standard: $M2_HOME/conf/settings.xml
  • En användares installation - standard: ${user.home}/.m2/settings.xml
Först- och sistnämnda settings.xml kallas även globala respektive användarspecifika inställningar. Om båda filer finns kommer deras innehåll slås samman, där användarspecifika settings.xml prioriteras.
