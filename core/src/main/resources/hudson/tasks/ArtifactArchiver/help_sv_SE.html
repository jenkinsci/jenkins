Arkiverar byggartefakterna (t.ex. zip- eller jar-filer för distribution) så att de kan laddas ned senare. Arkiverade filer kommer att vara tillgängliga från Jenkins webbsida.
Vanligtvis behåller Jenkins artefakter för ett bygge så länge som själva byggloggen behålls, men om du inte behöver gamla artefakter och hellre vill spara diskutrymme kan du göra det.

Observera att Maven-jobbtypen automatiskt arkiverar alla producerade Maven-artefakter. Alla artefakter som konfigurerats här kommer att arkiveras ovanpå det. Automatisk arkivering av artefakter kan inaktiveras under de avancerade Maven-alternativen.

Pipeline Snippet Generator genererar detta exempel när alla argument är "true" (vissa argument är "true" som standard): 
archiveArtifacts artifacts: '**/*.txt',
                   allowEmptyArchive: true,
                   fingerprint: true,
                   onlyIfSuccessful: true
