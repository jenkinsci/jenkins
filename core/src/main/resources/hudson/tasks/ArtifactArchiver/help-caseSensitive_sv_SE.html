Artefaktarkiveringen använder Ant org.apache.tools.ant.DirectoryScanner som är skiftlägeskänslig som standard. Om jobbet t.ex. producerar *.hpi-filer kommer mönstret "**/*.HPI" inte att hitta dem.

Detta alternativ kan användas för att inaktivera skiftlägeskänslighet. När detta avmarkeras kommer mönstren "**/*.HPI" och "**/cAsEsEnSiTiVe.jar" att matcha alla *.hpi-filer respektive en fil som heter caseSensitive.jar.
