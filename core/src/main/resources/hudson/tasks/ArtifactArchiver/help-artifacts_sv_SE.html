Du kan använda jokertecken som "modul/dist/**/*.zip". Se include-attributet för Ant-filuppsättningar för det exakta formatet -- förutom att "," ( komma ) är den enda avgränsaren som stöds. Baskatalogen är arbetsytan . Du kan bara arkivera filer som finns i din arbetsyta.

Här är några exempel på hur pipelines kan användas:

  • Hur man arkiverar flera artefakter från en specifik mapp: 
    archiveArtifacts artifacts: 'target/*.jar'

  • Hur man arkiverar flera artefakter med olika mönster: 
    archiveArtifacts artifacts: 'target/*.jar, target/*.war'

  • Hur man arkiverar flera kapslade artefakter: 
    archiveArtifacts artifacts: '**/*.jar'
