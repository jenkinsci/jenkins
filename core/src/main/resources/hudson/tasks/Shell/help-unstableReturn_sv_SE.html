Om detta anges kommer skalprogrammets slutkod att tolkas som ett instabilt byggresultat. Om slutskoden matchar värdet kommer byggresultatet ändras till "instabilt" och nästkommande steg återupptas. På Unix-liknande miljöer är det ett värde mellan 0-255. Värdet 0 ignoreras och gör inte bygget instabilt för att hålla standardbeteendet konsekvent.
