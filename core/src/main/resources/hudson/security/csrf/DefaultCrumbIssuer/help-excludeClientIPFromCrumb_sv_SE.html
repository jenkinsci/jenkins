Vissa HTTP-proxyer filtrerar bort information som standardutfärdaren för crumbs använder för att beräkna nonce-värdet. Om du har en HTTP-proxy mellan din webbläsarklient och Jenkins-server samt får 403-svar när du skickar ett formulär till Jenkins kan det här alternativet hjälpa. Genom att använda det här alternativet blir det lättare att förfalska nonce-värdet.
