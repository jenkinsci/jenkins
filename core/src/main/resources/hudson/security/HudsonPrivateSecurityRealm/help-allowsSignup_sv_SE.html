Detta alternativ låter användare skapa konton på egen hand via länken "registrera dig" längst upp till höger på sidan. Se till att inte bevilja betydande behörigheter till autentiserade användare eftersom alla i nätverket kommer att kunna få dem.

När den här kryssrutan är avmarkerad måste någon med administratörsrollen skapa konton.

Som standard använder Jenkins inte captcha-verifiering om användaren skapar ett konto själv. Om du vill aktivera captcha-verifiering får du installera ett insticksprogram med stöd för captcha, t.ex. Jenkins-insticksprogrammet JCaptcha .
