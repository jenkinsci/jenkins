I detta läge får alla inloggade användare full kontroll över Jenkins. De enda som inte kommer att ha det är anonym användare som bara får läsbehörighet.

Det här läget är användbart för att tvinga användare att logga in innan de vidtar åtgärder, så att du kan hålla koll på vem som har gjort vad. Den här inställningen kan också användas i offentliga Jenkins-miljöer där du bara låter betrodda användare ha användarkonton.
