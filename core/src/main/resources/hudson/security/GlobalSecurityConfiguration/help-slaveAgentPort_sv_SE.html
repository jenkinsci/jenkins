Om du inte använder ingående agenter rekommenderas du att inaktivera detta helt och hållet, vilket är installationens standardbeteende. Jenkins använder en TCP-port för att kommunicera med agenter som är anslutna ingående. Om du ska använda inkommande agenter kan du låta systemet att slumpmässigt välja en port vid start (detta undviker att störa andra program, inklusive andra Jenkins-instanser). Eftersom det är svårt för brandväggar att säkra en slumpmässig port kan du istället ange ett fast portnummer och konfigurera din brandvägg därefter.
