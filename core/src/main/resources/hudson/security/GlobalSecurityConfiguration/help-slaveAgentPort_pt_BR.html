O Jenkins usa uma porta TCP para se comunicar com os agentes agent lançados via JNLP. Normalmente esta porta é escolhida aleatóriamente para evitar colisões, mas isto tornaria difícil fazer a segurança do sistema. Se você não estiver usado agents JNLP, é recomendado desabilitar esta porta TCP. Alternativamente, você pode especificar um número de porta específico tal que você possa configurar seu firewall corretamente.
