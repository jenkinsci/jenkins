Detta fält fungerar tillsammans med fältet för proxyservern för att ange användarnamnet som används för att autentisera med proxyn.

Om denna proxy kräver Microsofts autentiseringsschema NTLM kan domännamnet kodas i användarnamnet med ett domännamn som prefix följt av en omvänt snedstreck "\" före användarnamnet, t.ex. "ACME\John Doo".
