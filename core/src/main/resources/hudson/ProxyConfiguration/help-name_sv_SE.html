Om din Jenkins-server är bakom en brandvägg och inte har direkt åtkomst till internet samt om JVM på din server inte är korrekt konfigurerad ( Se JDK-nätverksegenskaperna för mer information ) för att aktivera internetanslutning kan du ange namnet på HTTP-proxyservern i det här fältet för att låta Jenkins installera insticksprogram å dina vägnar. Observera att Jenkins använder HTTPS för att kommunicera med uppdateringscentret och ladda ned insticksprogram.

Om detta fält lämnas tomt kommer Jenkins försöka ansluta till internet direkt.

Kontrollera webbläsarens proxykonfiguration om du är osäker på värdet.
