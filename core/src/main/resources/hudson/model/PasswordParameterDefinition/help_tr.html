Yapılandırmaya bir şifre iletir. Burada girilen şifre, tıpkı bir string parametresi gibi ortam değişkeni olarak plain text formatında yapılandırma için kullanılabilir hale getirilir. Değer, Jenkins controller üzerinde şifrelenmiş olarak saklanır, Jenkins yapılandırmasındaki diğer şifrelerde olduğu gibi.
