Som standard kan byggen från detta projekt köras på alla tillgängliga agenter som är konfigurerade att acceptera nya byggen.

När det här alternativet är markerat har du möjligheten att säkerställa att byggen av detta projekt endast sker på en viss agent eller uppsättning av agenter.

Om ditt projekt t.ex. bara ska byggas på ett visst operativsystem, på datorer med specifika installerade verktyg eller till och med en specifik dator kan du begränsa det här projektet till att endast köras på agenter som uppfyller dessa kriterier.

Hjälptexten för fältet Etikettuttryck som visas när det här alternativet är markerat ger mer detaljerad information om hur man anger agentkriterier.
