Etiketler (diğer adıyla tag'ler) birden fazla agent'i mantıklı bir grup içinde birleştirmek için kullanılır.

Mesela, birden fazla Windows agent'iniz varsa ve bazı işlerinizin Windows üzerinde çalışması gerekiyorsa, bu durumda Windows agent'lerini 'windows' etiketi ile birleştirip, çalıştırmak istediğiniz işe, çalışacağı yer olarak 'windows' etiketini vermeniz yeterli olacaktır. Ayarlarınızı bu şekilde yaparsanız, istediğiniz işler, windows agent'lerinden başka yerde çalıştırılmayacaktır.
