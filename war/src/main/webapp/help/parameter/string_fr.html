Définit un paramètre textuel simple, où les utilisateurs peuvent entrer une valeur de type String. Il peut être utilisé dans un build comme variable d'environnement, ou dans d'autres endroits de la configuration, grâce à la substitution de variables.

Notez que, lorsque ce champ est exposé comme variable d'environnement, il sera toujours en majuscules (par exemple, le paramètre "foo" sera exposé sous le nom "FOO".)
