Defina um padrão (expressão regular) para checar se o nome de uma tarefa é válido ou não.

Forçando a checagem em tarefas que já existe, vai permitir que você reforce uma convenção - e.g. mesmo que os usuários não mudem o nome, A tarefa vai ser validada com o padrão definido em cada submissão e atualização até que o nome confirme.
Essa opção não afetará a execução de tarefas com nomes não complascentes. Isto apenas controla o processo de validação no momento de salvar as configurações de uma tarefa.
