Bu kısım, web uygulaması olan CVS repository tarayıcılarını belirlemek içindir. Bu alanları doldurarak Jenkins'ın, değişiklikler sayfasında ilgili linkleri oluşturmasını sağlayabilirsiniz. Her bir CVSROOT için iki farklı kısım vardır.

URL
Repository'deki bir dosyanın detaylarını gösteren URL
Fark URL
İki revizyon arasndaki farkları gösteren sayfanın URL'i

Aşağıdaki makro karakterler kullanılabilir.
%%P CVS altında dosyanın yolu, "foo/bar/Zot.java" gibi
%%r Eski versiyonun revizyonu (fark URL için)
%%r Yeni versiyonun revizyonu (fark URL için)
