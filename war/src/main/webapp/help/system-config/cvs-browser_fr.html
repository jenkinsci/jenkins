Spécifie les chemins du l'application web qui sert de navigateur de repository CVS. Ces valeurs permettent à Jenkins de générer des liens à partir des listes de changements vers ces pages. Il y a deux valeurs à fournir pour chaque CVSROOT.

URL
L'URL de la page qui affiche les détails d'un fichier particulier.
Diff URL
L'URL de la page qui affiche les détails des différences entre deux revisions.

Les caractères de macro suivants sont disponibles.
%%P Chemin vers le fichier sous CVS, du type "foo/bar/Zot.java"
%%r Numéro de révision de la version précédente (pour les Diff URLs)
%%r Numéro de révision de la nouvelle version du fichier (pour les Diff URLs)
