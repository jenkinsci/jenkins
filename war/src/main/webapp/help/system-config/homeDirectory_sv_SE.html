Man kan ändra Jenkins hemkatalog på olika sätt:

  • Konfigurera miljövariabeln JENKINS_HOME när man startar Jenkins.
  • Konfigurera Java-systemegenskapen JENKINS_HOME när man startar Jenkins.
Detta värde kan inte ändras medan Jenkins körs.
Det visas här för att hjälpa dig säkerställa att din konfiguration träder i kraft.
