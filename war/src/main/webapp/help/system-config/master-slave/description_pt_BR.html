Descrição opcional deste agente. Este informação é mostrada na tela de configuração do projeto.

Quando você tiver agentes que são diferentes dos outros, é útil usar este campo para explicar o que é diferente. Por exemplo, entrar "Agente Windows" neste campo permitiria aos donos do projeto escolher sempre construir em uma máquina Windows (por exemplo, se eles precisarem de uma ferramenta de construção específica do Windows.)
