Comprendre comment Hudson est utilisé est d'une grande aide pour guider les développements, particulièrement pour les projets open-source où les utilisateurs sont typiquement difficiles à connaitre. Quand cette option est activée, Hudson envoie périodiquement des informations sur votre utilisation de Hudson.

Plus précisément, il envoie les informations suivantes:

  • Votre version de Hudson
  • Pour le Hudson maître et pour chaque esclave, le type d'OS et le nombre d'exécuteurs
  • Les plugins qui sont installés et leurs versions
  • Le nombre de jobs pour chaque type de job dans votre installation d'Hudson

Cette information ne contient rien qui vous identifie ou qui nous permet de vous contacter (hors certaines informations intrinsèques à HTTP, comme votre adresse IP). Les données compilées de ces statistiques d'utilisation selon partagées avec la communauté.

N'hésitez pas à lire la discussion sur ce mécanisme et à contribuer aux débats.
